# web-
web开发课程实验一，实验主要目的在于掌握HTML5语义化标签及CSS基础语法
页面整体结构设计采用 Bootstrap 网格系统 + Flexbox 进行布局，页面划分为五个主要区域：
1.	导航栏 – 透明背景但使用深蓝色主题，包含品牌名、菜单项、登录/注册按钮；在手机视图下自动折叠为汉堡菜单。
2.	交互区（Hero） – 全宽背景图，居中显示标题、副标题和“马上学习”按钮。
3.	高级Web编程实验室 – 左图右文（PC端），手机端自动堆叠。
4.	双屏学习 – 右图左文（PC端），手机端同样堆叠。
5.	课程特色 – 三张卡片，展示“丰富的教学服务”、“多元的学习方式”、“实时互动教学”。
6.	页脚 – 深色背景，版权信息。
